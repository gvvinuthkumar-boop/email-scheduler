Email Scheduler Backend API:-

A simple backend application to schedule emails using Node.js, Express, and MongoDB.  
This project was built as part of a technical assignment to demonstrate backend development skills, REST API design, and database integration.

features
- Create (Schedule) an email
- Read a scheduled email by ID
- List all scheduled emails
- Update / Reschedule an email
- Delete a scheduled email
- MongoDB integration using Mongoose
- RESTful API architecture
  
technologies:-

- Backend: Node.js, Express.js
- Database: MongoDB
- ODM: Mongoose
- Testing Tool:Thunder Client (VS Code)
- Version Control: Git & GitHub
